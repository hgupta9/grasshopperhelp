SPM Integration Settings

Component Index > SPM > Integration > IntegrationS

Settings to configure a SPM vector field integration component

Inputs

NameIDDescriptionType
TolDistance from field where integration will be in effect. 0 will ignore the field entirelyNumber
IntrpRRadius to use in vector interpolation method. 0 to use closest vector onlyNumber
StopAThe winding angle is used to stop the integration if it winds around the set angle (radians). Used to stop the integration at singularities or orbitsNumber
LContForce lines to follow a straighter path if one is availableBoolean
FastUse a faster and less accurate integration method (Velocity-Verlet) instead of the default (Runge-Kutta)Boolean
DragDrag factor of the integrationNumber
AccelIntegration will contain a componenet of acceleration (the vectors will represent forces)Boolean

Outputs

NameIDDescriptionType
