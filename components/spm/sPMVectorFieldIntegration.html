SPM Vector Field Integration

Component Index > SPM > Integration > SPM Integration

Statically or dynamically integrates points travelling through a vector field

Inputs

NameIDDescriptionType
PList of points in the vector field (parallel to VT)Point
VTList of vectors or tensors in the vector field (parallel to P)Generic Data
SPPoints to simulate travelling through the vector fieldPoint
LTNumber of iterations for each particle to exist, by default these are respawned at their original start points when this time is upInteger
DList of dynamics to modify the vector field withGeneric Data
SList of settings objects to customize integration parametersGeneric Data
DIDynamically simulate the integration, one step at a time. If true, each run will integrate a series of particles for lifetime of LT. If false then the integration will run once and return all points at once. In this case LT is the number of steps to integrate.Boolean
RSReset the simulation to the starting point. This is only used when in dynamic integration mode (DI is true)Boolean

Outputs

NameIDDescriptionType
PTThe Travelling points after they have travelled [step] iterationsPoint
VTThe resultant vectors for each point during the simulationVector
