Vortex Dynamic

Component Index > SPM > Dynamics > Vortex

Creates vortices inside a vector field at specified points (at the origins of the planar inputs)

Inputs

NameIDDescriptionType
PlPlane for the vortex to operate on. The origin of the plane is the singularity point of the vortexPlane
hDistance from field where integration will halt. 0 to run until integration is completeNumber
kScalar multiple of the gravity force (positive for attraction, negative for repulsion)Number
aScalar multiple of the effect the spiral causes on the vector fieldNumber
eUse exact geodesic distances to calculate spiral path when on a surface, otherwise use a fast approximationBoolean
FTrue if you want a funnel instead of a vortex. A funnel will attract on one side and shoot out the otherBoolean
RTrue if you want to reverse the spiral when constrained to a surfaceBoolean

Outputs

NameIDDescriptionType
DParameters used to modify the vector field of the Vector Field Creater that this must be wired intoGeneric Data
