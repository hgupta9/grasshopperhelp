Create Chart

Component Index > LunchBox > LunchBox Data > Create Chart

Creates a saveable Winform chart.

Inputs

NameIDDescriptionType
NameThe name of the chartText
SeriesA list of Series names for the chart. (Should be a list equal to the number of tree branches used for values.Text
X-Axis LabelsA list of X-Axis Label names. (Should be equal to the number of items in a branch.)Text
ValuesA tree of numeric values. Branch order should correspond to the chart series names.Number
