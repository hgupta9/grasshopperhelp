Define Block

Component Index > Human > Creation > DefBlock

This component allows you to define or replace a block definition.

Inputs

NameIDDescriptionType
Block NameBNThe name of the block definitionText
GeometryGThe geometry that makes up the block.List of Geometry
AttributesAThe optional geometry attributes (like layer, color, etc)List of Generic
Block OriginBOThe base plane of the block definitionPlane
OverwriteOSet this to true in order to overwrite the block definition.Boolean

Outputs

NameIDDescriptionType
Block NameBNThe names of successfully defined blocksText
