CreateHatch

Component Index > Human > Creation > CreateHatch

Create Hatch Objects with specified pattern, scale, rotation, and attributes

Inputs

NameIDDescriptionType
Bake?bakeSet this value to true to bake the hatches into your active Rhino Document.Boolean
CurvescrvThe sets of curves to hatch. Curves in the same branch will be processed at the same time, producing internal voids where appropriate.List of Curve
Hatch Pattern NamenameThe name of the hatch to search for. If left blank or if the specified hatch cannot be found, a solid hatch will be used.List, "Solid" of Text
Hatch ScalescaleHatch scaleList, double of Number
Hatch RotationrotHatch rotation in radiansList, double of Number
AttributesattThe hatch's object propertiesList of Generic

Outputs

NameIDDescriptionType
geoThe surfaces or curves of the specified hatchGeometry
