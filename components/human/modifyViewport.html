Modify Viewport

Component Index > Human > Document Modifications > Mod Viewport

Use this component to restore a named view in a viewport, set its display mode, or set its camera.

Inputs

NameIDDescriptionType
Viewport NameNThe name of the viewport to modify. If left blank, active viewport will be used.Text
Named View to RestoreNVThe optional name of a named view to restore in this viewport.Text
Display ModeDMOptional Display mode to set this viewport to.Text
Camera LocationCLThe optional camera location point. This will override a named view.Point
Camera TargetCTThe optional camera target point. This will override a named view.Point
Camera ProjectionCPran optional boolean value for the camera projection. False for parallel, true for perspective.Boolean
Camera Lens LengthCLLOptional lens length for the perspective cameraNumber
