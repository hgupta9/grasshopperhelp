Prune Tree

Component Index > Sets > Tree > Prune Tree

Removes all branches from a Tree that carry a special number of Data items. You can supply both a lower and an upper limit for branch pruning.

Inputs

NameIDDescriptionType
TData tree to pruneGeneric Data
N0Remove branches with less than N0 items.Integer
N1Remove branches with more than N1 items (use zero to ignore upper limit).Integer

Outputs

NameIDDescriptionType
TPruned treeGeneric Data
