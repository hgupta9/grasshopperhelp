Modify Cross Sections (karamba)

Component Index > Karamba > Karamba Cross Section > Modify Cross Sections (karamba)

Modifies all types of cross sections.

Inputs

NameIDDescriptionType
CroSecCross section to modify on the fly or no input if modification shall take place in the model based on the name of the cross section.CrossSection
NameThe Type name serves to identify a cross section uniquely upon selection from a cross section table.Text
FamilyFamily name of cross section: defines a group of cross sections for cross section optimization.Text
Elem IdsList of identifier of elements to attach cross section to.Text
ecce-locVector of eccentricity in [cm] with respect to beam axis in local beam coordinates.Vector
HeightHeight [cm] of the cross-section given in centimeter if >0.Number
ThickWall-thickness [cm] of the cross-section given in centimeter if >0.Number
ACross section area [cm2] if >0.Number
AyEquivalent shear area [cm2] for shear in local Y-direction if >0.Number
AzEquivalent shear area [cm2] for shear in local Z-direction if >0.Number
IyyMoment of inertia [cm4] about the local Y-axis if >0.Number
IzzMoment of inertia [cm4] about the local Z-axis if >0.Number
IppPrimary or St. Venant torsional moment of inertia [cm4] about the local X-axis if >0.Number
CwWarping torsional constant [cm6] if >0.Number
Wely+Elastic resistance moment [cm3] about the local Y-axis for the extreme point on the positive local Z-axis if >0.Number
Wely-Elastic resistance moment [cm3] about the local Y-axis for the extreme point on the negative local Z-axis if >0.Number
WplyPlastic resistance moment [cm3] about the local Y-axis if >0.Number
Welz+Elastic resistance moment [cm3] about the local Z-axis for the extreme point on the positive local Y-axis if >0.Number
Welz-Elastic resistance moment [cm3] about the local Z-axis for the extreme point on the negative local Y-axis if >0.Number
WplzPlastic resistance moment [cm3] about the local Z-axis if >0.Number
WtMinimum torsional resistance moment [cm3] if >0.Number
alphaYBuckling coefficient about local Y-axis according to EC3 if >0.Number
alphaZBuckling coefficient about local Z-axis according to EC3 if >0.Number
alphaLTCoefficient for lateral torsional buckling according to EC3 if >0.Number

Outputs

NameIDDescriptionType
CroSecModified cross section or cross section modifier.CrossSection
