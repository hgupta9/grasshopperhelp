Vector Maps

Component Index > Mosquito > Mosquito Geo > Vector Maps

Extracts Vector maps of cities from OSM. Roads, Buildings etc (Building Heights are not available for all locations) - by Carson Smuts 2015

Inputs

NameIDDescriptionType
AreaThe area you wish to search forGeometry
AllRoadsLoads all roads WARNING takes a long time to loadBoolean
BuildingsLoads all Buildings WARNING takes a long time to loadBoolean
AmenitiesLoads all AmenitiesBoolean
Custom SearchLoad a custom data set for example see wiki.openstreetmap.org/wiki/Map_Features e.g. shop=clothingText
ReProjectUnskew the geometry and Project to MetersBoolean
CenterToWorldCenters all the geometry to the center of world = 0,0,0Boolean
ServerTwoIf there is a server problem, try thisBoolean

Outputs

NameIDDescriptionType
RoadLinesThe CenterLine of RoadsCurve
RoadInfoRoad InfoText
BuildingLinesThe Building footprintCurve
BuildingInfoBuilding InfoText
BuildingHeightsBuilding Heights if availableNumber
AmenityLinesAmenity footprintsCurve
AmenityInfoAmenity InfoText
AmenityPointsAmenity pointsPoint
AmenityPointInfoAmenity Point InfoText
CustomLinesCustom Search footprintsCurve
CustomInfoCustom Search InfoText
CustomPointsCustom Search pointsPoint
CustomPointInfoCustom Search Point InfoText
BenchMarkThe Object NameText
ProjPointUse this with the Project plugin so your geometry scale properlyPoint
CenterPointUse this with the Project plugin so your geometry overlaps correctlyPoint
