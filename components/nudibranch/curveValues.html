CurveValues

Component Index > NudiBranch > Attractors > CrvValues

Generates numerical values for any given pointlist, being affected by any number of curve attractors.

Inputs

NameIDDescriptionType
PointsPPoint List to affectTree of Point
CurvesCCurve AttractorsList of Curve
RangeRCurve attractor's unique range (Leave one single value for global)List, (double of Number
DistributionDSelect a force distribution:(0=Linear,1=parabola,2=sine,3=noise,4=sinc,5=hyperbolic,6=square root,7= power^3,8=radial random,9=cross sines,10= cos^2) (Leave one single value for global)List, 1 of Integer
Asc/DescAsToggle between Ascending/Descending valuesBoolean
Min/MaxMiToggle between Minimum/Maximum valuesBoolean
ForceFGlobal force for all the curve attractorsNumber

Outputs

NameIDDescriptionType
ValuesVThe resulting valuesTree of Number
