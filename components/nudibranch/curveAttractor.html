CurveAttractor

Component Index > NudiBranch > Attractors > CrvAttractor

Generates an iteratively defined new point/vector list for any given, curve attractor affected, population of points.

Inputs

NameIDDescriptionType
PointsPPoint List to affectTree of Point
CurvesCCurve AttractorsList of Curve
RangeRAttractor's unique range (Leave one single value for global)List, double of Number
DistributionDSelect a force distribution:(0=Linear,1=parabola,2=sine,3=noise,4=hyperbolic,5=square root,6=power^3) (Leave one single value for global)List, 1 of Integer
Attraction/RepulsionReToggle between attraction and repulsion (Leave one single value for global)List, true of Boolean
Asc/DescAsToggle between Ascending/Descending valuesBoolean
Min/MaxMiToggle between Minimum/Maximum valuesBoolean
ForceFGlobal force for all the attractorsNumber
IterationsItNumber of iterationsInteger

Outputs

NameIDDescriptionType
LengthLThe lenght of the vectors sum Tree of Number
NewPointsPThe generated point list per iterationTree of Point
NewVectorsVThe generated vector list per iterationTree of Vector
