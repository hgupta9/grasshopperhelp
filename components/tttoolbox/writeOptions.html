Write Options

Component Index > TT Toolbox > TT Toolbox Excel > Write Options

Option settings for writing data to Excel.

Inputs

NameIDDescriptionType
[FP]File path pointing to an excel file which will be opened and written to. Only .xlsx file types will be recognized. If the file does not exist and the path is valid, a new file will be created and saved after writing data.Text
[C/O]Set to false to partially overwrite data in excel and maintain any formatting in the worksheet. True clears all data and formatting in the sheet, then writes new data.Boolean
[Save]Set to true to save the excel file after writing data. A valid file path must be specified in order for a file to be saved.Boolean
[Close]Set to true to close the file after writing data. If true, Save after write will be set to true automatically. Use this setting to create a series of files by animating a slider to avoid opening a bunch of excel instnaces.Boolean
[Cs]Cell to start writing columns in.Integer
[Rs]Row to start writing columns in.Integer
[FC]A list of 2 or 3 colors to define the conditional formatting color range. Conditional formatting will be added to the tree of data if colors are specified.Colour

Outputs

NameIDDescriptionType
OutText output for errors, messages, debugging, etc.Text
WOOptions string to input into Write To Excel component.Text
